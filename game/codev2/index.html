Help

Welcome to Game of Pac. Use the 'WASD' or arrow keys to recreate your sequence, then roll the dice. The first one to 30 or more points wins the game!

Additional rules:

  • You will not get points if you roll a bomb (one).
  • Both players' score will return to 0 if either rolls a cherry bomb (double ones). Good luck!
